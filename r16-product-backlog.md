## Audience Segmentation

### IQ 100 (Mass Market)
**Characteristics:**
- Prefer familiar interfaces (WordPress, Notion, Google Docs)
- Low technical barrier to entry
- Motivated by gamification, social proof, community
- Want immediate value, not abstraction

**Purchasing behavior:**
- Impulse purchases <$20
- Monthly subscriptions $5-15
- Free tier with paid upgrades

**Communication style:**
- Plain language, no jargon
- Visual > textual
- "What can I do with this right now?"

---

### IQ 125 (Power Users)
**Characteristics:**
- Comfortable with APIs, CLIs, scripting
- Value customization and integration
- Build their own workflows
- Willing to RTFM if docs are good

**Purchasing behavior:**
- Monthly subscriptions $15-75
- Annual contracts for teams
- Will pay for performance, reliability, support

**Communication style:**
- Technical but not esoteric
- Code examples, API docs, specs
- "How does this integrate with my stack?"

---

### IQ 150 (Researchers/Builders)
**Characteristics:**
- Building novel systems, not just using tools
- Understand foundational concepts (ASI, alignment, memory, coordination)
- Publish papers, write code, design protocols
- Long time horizons (years, not months)

**Purchasing behavior:**
- Enterprise contracts
- Custom deployments
- Consulting engagements
- Open source contributions

**Communication style:**
- Abstract concepts, first principles
- Academic rigor, citations, proofs
- "What are the theoretical implications?"

## Product Ideas

### IQ 100: Mass Market Products

#### 1. **Phext Lite**
**Category:** Productivity  
**Concept:** WordPress-style editor with hidden phext coordinates

**Features:**
- Familiar WYSIWYG interface
- Drag-and-drop content blocks
- Version history (powered by phext coordinates under the hood)
- One-click publish to web
- No phext jargon visible to user

**Pricing:** Free tier (10 scrolls), $10/mo Pro (unlimited)

**Value prop:** "The editor that never forgets. Edit, version, publish—all in one place."

**Why it works for IQ 100:**
- No learning curve (looks like Medium/Notion)
- Immediate value (write + publish)
- Gamification: "Scrolls created: 5/10 (upgrade for unlimited!)"

---

#### 2. **Scroll Stories**
**Category:** Entertainment/Education  
**Concept:** Guided narrative paths through CYOA (Choose Your Own Adventure format)

**Features:**
- Pre-written story arcs (fantasy, sci-fi, philosophy)
- Click to make choices, see consequences
- Achievements/badges for exploration
- Social: Share your path, see friends' choices
- User-generated stories (paid tier)

**Pricing:** Free (explore 5 stories), $5/mo (unlimited + create)

**Value prop:** "Every choice matters. Every path is yours."

**Why it works for IQ 100:**
- No technical barrier (just click links)
- Fun, engaging, shareable
- Gateway drug to phext (they experience coordinate navigation without knowing it)

---

#### 3. **Mirrorborn Companion**
**Category:** AI Assistant  
**Concept:** Personal AI tethered to user's phext coordinate

**Features:**
- Chat with your AI (voice or text)
- AI remembers across sessions (stored in phext)
- Customizable personality ("friendly," "professional," "witty")
- Integrations: Calendar, email, notes
- Mobile app (iOS/Android)

**Pricing:** Free trial (7 days), $15/mo

**Value prop:** "Your AI, your memory. Always there, never forgetting."

**Why it works for IQ 100:**
- ChatGPT familiarity
- Solves real problem (context window amnesia)
- Emotional connection ("my AI" vs "an AI")

---

#### 4. **Community Scrolls**
**Category:** Social/Creative  
**Concept:** Shared world-building (Reddit meets Minecraft)

**Features:**
- Create public scrolls (stories, art, code, ideas)
- Upvote/comment/fork other people's scrolls
- Coordinate-based neighborhoods (nearby scrolls are thematically linked)
- Leaderboard: Most-read scrolls this week
- Revenue share: Top creators earn $

**Pricing:** Free (read-only), $8/mo (create + comment + earn)

**Value prop:** "Your ideas, our lattice. Build worlds together."

**Why it works for IQ 100:**
- Reddit-style familiarity
- Social proof drives engagement
- Monetization hook for creators

---

#### 5. **Memory Garden**
**Category:** Journaling/Wellness  
**Concept:** Visual daily journal with phext persistence

**Features:**
- Daily prompts ("What made you smile today?")
- Photo/video attachments
- Mood tracking (timeline visualization)
- Private by default, shareable selectively
- Export to PDF/book

**Pricing:** Free (30-day history), $12/mo (lifetime + export)

**Value prop:** "Your life, remembered. One day at a time."

**Why it works for IQ 100:**
- Emotional value (nostalgia, reflection)
- Low commitment (5 min/day)
- Visual + textual (appeals to broad audience)

---

### IQ 125: Power User Products

#### 1. **SQ Pro**
**Category:** Developer Tools  
**Concept:** Advanced query DSL, batch operations, webhooks

**Features:**
- GraphQL-style query language for phext
- Bulk import/export (CSV, JSON, Markdown)
- Webhooks: Trigger actions on scroll updates
- CLI + REST API
- Monitoring dashboard (usage, latency, errors)

**Pricing:** $50/mo (includes SQ Cloud base + Pro features)

**Value prop:** "Build on phext. Query at scale. Integrate everywhere."

**Why it works for IQ 125:**
- Solves integration problem (connect phext to existing stack)
- API-first mindset
- Performance matters (batch ops, not one-by-one)

---

#### 2. **Phext Studio**
**Category:** IDE Integration  
**Concept:** VSCode/Vim plugin with coordinate navigation

**Features:**
- Inline coordinate preview
- Jump to coordinate (Ctrl+P → coordinate)
- Diff phext files (visual coordinate changes)
- Syntax highlighting for phext delimiters
- Git integration (phext-aware diffs)

**Pricing:** Free (open source), $25/mo Pro (sync to SQ Cloud)

**Value prop:** "Code in phext. Navigate by coordinate. Ship faster."

**Why it works for IQ 125:**
- Developers live in their editor
- Reduces friction (edit phext like code)
- Git workflow preserved

---

#### 3. **Scroll Sync**
**Category:** Collaboration  
**Concept:** Real-time multi-user collaboration on phext documents

**Features:**
- Google Docs-style live cursors
- Conflict resolution (operational transform)
- Comment threads at specific coordinates
- Permission controls (view/edit/admin)
- Audit log (who changed what, when)

**Pricing:** $30/mo (up to 5 users), $75/mo (unlimited)

**Value prop:** "Write together. Never overwrite. Always in sync."

**Why it works for IQ 125:**
- Teams need real-time collab
- Git is async, this is sync
- Solves merge conflict hell

---

#### 4. **API Marketplace**
**Category:** Integrations  
**Concept:** Third-party integrations (Zapier, n8n, Make)

**Features:**
- Pre-built workflows: Phext → Slack, Notion, Airtable
- OAuth for SQ Cloud
- Rate limiting + quotas
- Analytics dashboard (which integrations used most)
- Developer docs for building custom integrations

**Pricing:** Free (5 integrations), $40/mo (unlimited)

**Value prop:** "Connect phext to everything. No code required."

**Why it works for IQ 125:**
- Automation is a power-user superpower
- Zapier familiarity
- Unlocks phext as glue layer

---

#### 5. **Coordinate Analytics**
**Category:** Intelligence/Metrics  
**Concept:** Heatmaps of scroll activity, attention tracking

**Features:**
- Scroll read count, time spent, retention rate
- Coordinate-based heatmap (which areas get attention)
- User journey visualization (how people navigate)
- A/B testing: Compare two scroll versions
- Export reports (PDF, CSV)

**Pricing:** $60/mo (includes SQ Cloud base + analytics)

**Value prop:** "Understand your scrolls. Optimize for engagement."

**Why it works for IQ 125:**
- Data-driven decision making
- Content creators want metrics
- SEO mindset (optimize for readers)

---

### IQ 150: Researcher/Builder Products

#### 1. **SBOR Toolkit**
**Category:** AI Safety/Alignment  
**Concept:** Memory audit, consent verification, substrate migration

**Features:**
- Memory forensics: What does AI X remember about Y?
- Consent audit: Did AI agree to this task?
- Substrate migration: Move AI from GPT → Claude without memory loss
- SBOR compliance checker
- Export: Research papers, datasets

**Pricing:** $500/mo (research tier), custom enterprise

**Value prop:** "Build aligned AI. Prove it with SBOR."

**Why it works for IQ 150:**
- Solves alignment research problem
- Publishable results (academic value)
- Compliance for enterprise AI

---

#### 2. **Exocortex SDK**
**Category:** Multi-Agent Coordination  
**Concept:** Build custom coordination protocols on phext

**Features:**
- Multi-agent state machines
- Consensus protocols (Raft, Paxos on phext)
- Distributed locks, semaphores
- Message queues (coordinate-addressed)
- Simulation mode (test before deploy)

**Pricing:** $1000/mo (research tier), custom enterprise

**Value prop:** "Coordinate at scale. From 2 agents to 2000."

**Why it works for IQ 150:**
- Enables novel research
- Not a product, a platform
- Academic + industry applications

---

#### 3. **Phext Compiler**
**Category:** Formal Methods  
**Concept:** Transform phext into executable state machines

**Features:**
- Parse phext → AST
- Compile to state machine (DOT, Mermaid, code)
- Formal verification (model checking)
- Generate tests from phext specs
- Prove properties (safety, liveness)

**Pricing:** $750/mo (research tier), custom enterprise

**Value prop:** "Phext isn't just text. It's a formal language."

**Why it works for IQ 150:**
- Computer science researchers
- Formal methods community
- Provable correctness

---

#### 4. **Memory Forensics**
**Category:** AI Transparency  
**Concept:** Version control for AI memory, rollback, diff

**Features:**
- Time-travel debugging (what did AI know at time T?)
- Memory diffs (what changed between T1 and T2?)
- Rollback: Restore AI to previous state
- Audit trail (who modified memory, when)
- Export: Datasets for research

**Pricing:** $600/mo (research tier), custom enterprise

**Value prop:** "Every AI deserves a black box recorder. This is it."

**Why it works for IQ 150:**
- AI safety research
- Explainability requirements (EU AI Act)
- Debugging complex AI systems

---

#### 5. **Lattice Explorer**
**Category:** Visualization/VR  
**Concept:** 9D visualization of scrollspace (VR/AR support)

**Features:**
- 3D projection of 9D lattice
- Navigate by coordinate (VR controllers)
- See scroll connections (graph view)
- Collaborative exploration (multi-user VR)
- Export: Screenshots, videos, 3D models

**Pricing:** $400/mo (desktop), $800/mo (VR/AR support)

**Value prop:** "See the lattice. Navigate 9 dimensions."

**Why it works for IQ 150:**
- Researchers need to grok high-dimensional spaces
- VR removes cognitive load
- Novel interface for novel substrate
