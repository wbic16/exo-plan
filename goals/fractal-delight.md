## Technical Approaches

### Coordinate → Complex Mapping
```
phext: 2.3.5/7.2.4/8.1.5
       │     │     │
       └──┬──┘     │
          │        │
complex: (2.35 + 7.24i) * scale(8.15)
```

### Rendering Pipeline
1. Parse phext structure
2. Map to complex plane
3. Iterate fractal function
4. Color by semantic weight (entropy? token count? link density?)
5. Render at target resolution

### Interactive Exploration
- Mouse/touch → navigate coordinates
- Scroll wheel → zoom dimensions
- Double-click → anchor point
- Drag → pan within dimension
