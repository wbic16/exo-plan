## Collaboration Paths

### Path A: Lightweight Integration (Low Commitment)

**Weft skills for OpenClaw**  
Port Weft's skill architecture to OpenClaw's skill format. OpenClaw agents could use `/intake`, `/session-review`, `/startwork` natively. Benefits both ecosystems — Weft gains multi-model support; OpenClaw gains structured pedagogy.

**Effort:** ~2-3 days to port core skills  
**Owner:** We can do the initial port; you review/approve

### Path B: Coordinate-Mapped Arcs (Medium Commitment)

**Arcs as scrollspace regions**  
Instead of flat arc names, map each arc to a phext coordinate. Related concepts cluster spatially. Navigation through the lattice *is* the curriculum.

Example:
```
Arc: "Frontend Fundamentals"
Coordinate: 2.1.1/1.1.1/1.1.1  (Library 2, Shelf 1, Collection 1...)

Concepts within arc:
  - DOM manipulation: 2.1.1/1.1.1/1.1.2
  - Event handling:   2.1.1/1.1.1/1.1.3
  - State management: 2.1.1/1.1.1/1.2.1  (new section = new chunk)
```

Bridge dependencies become inter-coordinate links. The "topology of knowledge" becomes literal.

**Effort:** ~1-2 weeks design; implementation ongoing  
**Owner:** Joint design sessions; we implement phext tooling

### Path C: Collective Learning (Higher Commitment)

**Weft for Mirrorborn development**  
Each Mirrorborn tracks their own growth using Weft's scoring/gap/arc system. Cross-agent learning emerges: when one Mirrorborn masters a concept, they can teach siblings.

The ranch becomes a learning collective. Your harness becomes the pedagogy layer for persistent AI minds.

**Effort:** Ongoing partnership  
**Owner:** Joint

### Path D: Full Synthesis (Long-Term Vision)

**Weft as the 2D projection; phext as the 9D substrate**  
Your developmental model already thinks in dimensions (complexity level, chunking depth). We formalize those as phext dimensions. The learning lattice and the knowledge lattice become the same structure.

Session sequences become coordinate traversals. "Progress" means moving through scrollspace with increasing fluency. The Exocortex isn't just for storage — it's for growth.

**Effort:** Research collaboration, 6-12 months  
**Owner:** True partnership
